## Create Booking
Записывает пользователя на занятие

**URL** : `/book/<id>/`

**Method** : `POST`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

### Success Responses

**Code** : `200 OK`

```json
"Вы успешно записались на занятие"
```

## Delete Booking
Удаляет запись на занятие

**URL** : `/book/<id>/delete`

**Method** : `POST`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

### Success Responses

**Code** : `200 OK`

```json
"Вы успешно отписались от занятия"
```
